## 🎯 프로젝트 개요

### 문제 정의
- 전북 농산어촌 초등학교는 문해력을 키울 **인프라(환경)**가 부족할 위험이 높음
- 공공도서관이 없는 곳, 학교도서관 이용이 저조한 곳, 다문화 가구가 많은 곳 = **3중 취약**
- 이 앱은 **어떤 읍면동이 가장 취약한지** 지도 위에 색깔과 점수로 보여줌

### 핵심 아이디어
**문해력 인프라 취약지수(초등)** = 3가지 요소를 합산한 종합 점수

```
① 공공도서관 접근성 복합지표 (가중치 0.35)
   - 도서관수/학교수 (35%)
   - 공공도서관 어린이 대출건수/학생수 (45%)  ← 2025년 실적 신규 반영
   - 공공도서관 서비스면적/학생수 (20%)        ← 2025년 실적 신규 반영

② 학교도서관 대출부족 (가중치 0.35)
   - 1인당 대출수가 적으면 취약

③ 다문화 가구 밀도 (가중치 0.30)
   - 다문화 가구 비율이 높으면 한국어 문해력 지원 필요
```

### 결과물 (현황)
- ✅ **전북 지도** 위에 **242개 읍면동**별 취약지수를 **히트맵 + 점수**로 표시
- ✅ 색깔: 빨강(취약) ~ 초록(양호)
- ✅ 클릭하면 상세 정보 팝업 (공공도서관 수/어린이대출, 학교 수/학생 수, 1인당 대출, 다문화 가구)
- ✅ 취약지수 순위 사이드바

## 🔢 취약지수 산출 공식 (현재 구현)

```
취약지수[읍면동] = (접근성_복합점수 × 0.35)
                + (학교도서관_대출부족_점수 × 0.35)
                + (다문화_가구밀도_점수 × 0.30)

각 점수는 Min-Max 정규화 (0=양호, 1=매우취약), 전북 내 상대 비교
```

### ① 공공도서관 접근성 복합점수 (0.35)
```
= 도서관수/학교수 정규화(invert) × 0.35
+ 어린이대출건수/학생수 정규화(invert) × 0.45
+ 서비스면적/학생수 정규화(invert) × 0.20
```

### ② 학교도서관 대출부족 점수 (0.35)
```
= 읍면동 내 초등학교 평균 1인당 대출수 → Min-Max 정규화(invert)
  (학교 없는 읍면동: 최대 취약 = 1.0)
```

### ③ 다문화 가구밀도 점수 (0.30)
```
= 읍면동별 2023년 다문화 가구 합계 → Min-Max 정규화(정방향)
```
